About

YT CSV Playlist Manager is a lightweight, privacy-focused browser extension that allows users to create and manage YouTube playlists using simple CSV files stored locally in the browser. Instead of relying on the YouTube API or a Google account, the extension builds temporary playlist queues directly from saved video links.

The tool enables users to quickly add the currently open YouTube video to a playlist, export their playlist to a CSV file for backup or editing, and import different CSV files to switch between playlist collections. All playlist data is stored locally using the browser's storage system, ensuring that no information is sent to external services.

This project was developed as a small web utility to explore browser extension development and data portability through simple spreadsheet-based workflows.
